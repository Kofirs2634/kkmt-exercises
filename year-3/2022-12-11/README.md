## `brackets.py`
В строке, содержащей программный код на C-подобном языке, проверить правильность расстановки скобок. Если все верно, вывести надпись `Success`, в противном случае вывести индекс первого неверного символа.

Пример:
```
((()[][]{{{[]}[]}}[()]))
Success
```
```
(([][]{}}))
9
```
